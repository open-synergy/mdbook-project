Odoo - Project Management

Membuat Project Secara Manual

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Project -> Project -> Projects. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Isi Project Name. Harus diisi.
  2. Aktifkan Use Task jika dibutuhkan.
  3. Aktifkan Timesheet jika dibutuhkan.
  4. Aktifkan Issue jika dibutuhkan.
  5. Isi Project Manager. Tidak harus diisi.
  6. Isi Customer. Tidak harus diisi.
  7. Abaikan Project Template. Tidak harus diisi.
  8. Isi Email Alias. Tidak harus diisi.
  9. Pilih Incoming Emails Create. Abaikan jika isian Email Alias kosong.
  10. Pilih Accept Emails From. Abaikan jika isian Email Alias kosong.
  11. Buka tab Description
  12. Isi Description. Tidak harus diisi.
  13. Buka tab Team
  14. Tambahkan/Hapus Project Team. Ulangi langkah ini sampai Project Team sesuai dengan keinginan.
  15. Buka tab Other Info.
  16. Pilih Privacy/Visibility. Harus diisi.
  17. Pilih Currency. Tidak harus diisi.
  18. Pilih Parent. Harus diisi.
  19. Buka tab Project Stages.
  20. Tambah/Hapus Project Stages. Ulangi langkah ini sampai Project Stages sesuai keinginan.
  21. Pilih Timezone. Tidak harus diisi.
  22. Pilih Start Schedule Based On. Harus diisi. Lanjutkan ke langkah ke-23 apabila isian sama dengan Manual. Lanjutkan ke langkah ke-24 apabila isian tidak sama dengan Manual.
  23. Isi Manual Baseline Start. Lanjutkan ke langkah ke-28.
  24. Pilih Project Based Schedule.
  25. Pilih Task Based Schedule.
  26. Pilih Baseline Start Offset.
  27. Pilih Baseline Start Offset UoM.
  28. Pilih Finish Schedule Based On. Harus diisi. Lanjutkan ke langkah ke-30 apabila isian sama dengan Manual. Lanjutkan ke langkah ke-31 apabila isian tidak sama dengan Manual.
  29. Isi Manual Baseline Finish. Lanjutkan ke langkah ke-35.
  30. Pilih Project Based Schedule.
  31. Pilih Task Based Schedule.
  32. Pilih Baseline Finish Offset.
  33. Pilih Baseline Finish Offset UoM.
  34. Klik tombol Save pada bagian atas-kanan form.
